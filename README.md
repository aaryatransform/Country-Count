# Country Count

A simple Python script to count countries from a dataset or list.

## Usage

1. Clone this repository:
   ```sh
   git clone <repo-url>
   ```
2. Run the script:
   ```sh
   python country_count.py
   ```

## Requirements

- Python 3.x

If your script uses any external libraries, add them to a `requirements.txt` file.
